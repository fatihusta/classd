Skins cookbook

The administration client and user facing pages can be skinned.

The skin has a folder structure described below. This must be zipped in a single file.To use the skin it must be uploaded on the server.

To upload a skin in the server open the administration client, go to Config tab and open Administration window. In Administration go to Skins tab and the section "Upload an new skin" browse for the zipped skin file and press Upload. The skin will be uploaded and it will become available.

Selecting administration client skin

Go to Config tab and open Administration window. Go to skin tab and form Administration skin section choose the desired skin and press save. The client gui will be displayed using the selected skin

User pages skin

User pages skin can be selected when Branding Manager commercial Add-on is enabled

When Branding Manager is enabled a tab called Branding is displayed in Administration window. To change the user pages skin go to Branding tab and form Block Page Skin desired skin and press save. The user facing pages will be displayed using the selected skin

Skin Structure:

The folder structure of a skin is: 
skin.xml - skin information file 
css/admin.css - administration client styles
   /user.css  - user pages styles
   /ext-skin.css - ExtJs skin styles


images/admin/* - administration client images
      /user/*  - user pages images
      /<extJs-images-folder> - the folder containing images used by ExtJs skin

skin.xml file

<?xml version="1.0" encoding="UTF-8"?>
<skin>
  <name>skinName</name>
  <displayName>Skin Display Name</displayName>
  <adminSkin>true</adminSkin>
  <userFacingSkin>true</userFacingSkin>
</skin>

name is skinName
displayName will be displayed in the skin chooser select box in Administration
adminSkin - true/false - is an administration client skin
userFacingSkin - true/false - is an user pages skin

ExtJs skin

ExtJs v2.2 framework is used in adminstration client for most components: windows, panels, tabs, buttons, tables.
ExtJs has a themes mechanism. It has some built in themes and alse you can find user themes can be defined: See http://extjs.com/learn/Ext_Extensions#User_Themes
The Ext Js themes file must be named ext-skin.css.
